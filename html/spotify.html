Spotfiy Clone

This is a Spotify clone aimed to emulate the styling and core functions of Spotify utilizing Spotify's API

Summary
My goal for this project was to be able to incorporate the Spotify API into a client side web player. The functionalities of this project include recent history associated to your spotify account, search, and the ability to view your playlists.
Role
Frontend, UX / UI
